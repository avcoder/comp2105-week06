.fadeIn  .fadeOut

Fade animations involve rapidly making changes to an element's CSS opacity property.

// Hide all paragraphs using a fade out animation over 1.5 seconds
$("p").fadeOut(1500);


// Show all hidden divs using a fade in animation over 0.75 seconds
$( "div.hidden" ).fadeIn( 750 );

.slideUp  .slideDown

Slide animations are accomplished by rapidly making changes to an element's CSS height property.

// Hide all paragraphs using a slide up animation over 0.8 seconds
$( "p" ).slideUp( 800 );
  
// Show all hidden divs using a slide down animation over 0.6 seconds
$( "div.hidden" ).slideDown( 600 );

Instead of passing in milliseconds,
you can also use:


  • "slow"

  • "normal"

  • "fast"

// Slowly hide all paragraphs
$( "p" ).fadeOut( "slow" );
  
// Quickly show all divs that have the hidden style class
$( "div.hidden" ).slideUp( "fast" );

.hide  .show

Hiding an element sets its CSS display property to none; the content will have zero width and height. You may have noticed a combination of slide and fade effects when showing and hiding content.

// Instantaneously hide all paragraphs
$( "p" ).hide();
  
// Instantaneously show all divs that have the hidden style class
$( "div.hidden" ).show();

.toggle

jQuery can also let you change a content's visibility based on its current visibility state. .toggle() will show content that is currently hidden and hide content that is currently visible.

// Instantaneously toggle the display of all paragraphs
$( "p" ).toggle();
  
// Slowly toggle the display of all images
$( "img" ).toggle( "slow" );
  
// Toggle the display of all divs over 1.8 seconds
$( "div" ).toggle( 1800 );

.slideToggle   .fadeToggle

.toggle() will use a combination of slide and fade effects, just as .show() and .hide() do. You can toggle the display of content with just a slide or a fade using .slideToggle() and .fadeToggle().


// Toggle the display of all ordered lists over 1 second using slide up/down animations
$( "ol" ).slideToggle( 1000 );
  
// Toggle the display of all blockquotes over 0.4 seconds using fade in/out animations
$( "blockquote" ).fadeToggle( 1000 );

.stop :animated

.stop() will immediately terminate all animations running on the elements in your selection. Because :animated is a jQuery extension and not part of the CSS specification...first select the elements using a pure CSS selector, then use .filter(":animated").


// Create a button to stop all animations on the page:
$( "button" ).click(function() {
  $( "body *" ).filter( ":animated" ).stop();
});

Doing something after animation completes

A common mistake is assuming that the next method in your chain will wait until the animation runs to completion.


// Fade in all hidden paragraphs; then add a style class to them (not quite right)
$( "p.hidden" ).fadeIn( 750 ).addClass( "lookAtMe" );

To defer an action until after an animation has run to completion, you need a callback function as the second argument 

                  
                    

// Fade in all hidden paragraphs; then add a style class to them (correct with animation callback)
  $( "p.hidden" ).fadeIn( 750, function() {
      // this = DOM element which has just finished being animated
      $( this ).addClass( "lookAtMe" );
  });

.animate( properties [, duration ] )

can only use numeric CSS properties in .animate(); Duration is specified in milliseconds. Directional properties (top, right, bottom, left) have no discernible effect on elements if their position style property is static, which it is by default.


$("#lime1").animate({
  left: "+=100"
}, 1000);

What happens if you just say left: "100" ?

.animate( properties [, duration ] [, easing] )

The only easing that jQuery has are swing (default), and linear. More easing functions are available with the use of plug-ins.


  $("#lime2").animate({
    left: "+=100"
  }, 1000, "linear");

What happens if you just say left: "100" ?





Easing plugins : jQuery UI's .effect()

if jQuery UI library is included, then you can use .effect() 


$('#golime3').click(function () {
  $("#lime3").effect("bounce", "slow");
});

$('#golime3b').click(function () {
  $("#lime3b").animate({left: 100}).effect("bounce", "slow");
});

Robert Penner's jquery.easing Plugin

to use these custom plugins you must include the library then use it in .animate() in the easing parameter. 


  $('#golime4').click(function () {
    $("#lime4").animate({
      left: "+=100"
    }, 1000, "easeOutElastic");
  });

.animate() has callbacks

.animate( properties [, duration ] [, easing ] [, complete ] ) 

  $("#lime5").animate({
    left: "+=100"
  }, 1000, function () {
    alert('animation completed');
  });
